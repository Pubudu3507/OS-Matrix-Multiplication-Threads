# OS-Matrix-Multiplication-Threads

## Requirements

### Pthread (C Code):
- GCC (Linux or Windows with MinGW)
- Pthreads library

### Java threads (JavaCode):
- Java JDK (8 or above)


##  How to Run

### Pthreads(c) 

```terminal
gcc pthread_matrix.c -o matrix -lpthread
./matrix



### Java thread (java)

```terminal
javac javathreads_matrix.java
java javathreads_matrix
